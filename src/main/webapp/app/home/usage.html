  1. Export your issues to xml

    1. Go to Jira -> Search for issues
    2. Search the issues you would like to add to the report
      1. (Optional) You can search your issues using the filters, but usually I prefer to create my own search:
      2. For creating a customize search, click on Advanced:

    4. Upload the file into the application
